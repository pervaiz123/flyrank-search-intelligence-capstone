# FlyRank Search Intelligence Capstone

## Project Title

Search Opportunity Intelligence Engine: Identifying High-Potential Pages for Ranking Growth

## Research Question

Can search performance metrics be used to identify pages with the highest opportunity for improving search visibility and discoverability?

## Objective

Build a repeatable machine learning and analytics workflow that analyzes search performance data, validates findings, and produces ranked content recommendations.

## Deliverables

* Research Question
* Safe Data Contract
* Exploratory Data Analysis (EDA)
* Baseline Model
* Machine Learning Model
* Validation & Leakage Audit
* Ranked Action Engine
* Charts & Visualizations
* Research Notes
* Public-Safe Case Study
* Demo Presentation
